# 🚀 JS Intro Basics: Mission Practices

*Created by Ms. Liu for Computer Science and Mathematics students.*

Welcome to your very first JavaScript adventure! This project is designed for absolute beginners to bridge the gap between reading about code and actually writing it. 

## 🎯 What You Will Learn

This project is broken down into three main stages, which you can follow directly in the `index.html` and `script.js` files:

* **Stage 1: Inline JavaScript (Missions 1-5)** Writing code directly inside the HTML `<button>` tags to change text, modify CSS styles, hide elements, and turn a lightbulb on and off!
  
* **Stage 2: JavaScript Functions (Day 1)**
  Learning to organize our code like real developers by moving the logic into a separate `script.js` file and wrapping it in reusable functions like `changeText()` and `lightson()`.

* **Stage 3: Variables & Calculators (Day 2 & 3)**
  Using `let` to store numbers, getting user input from HTML boxes, converting text to Numbers, and performing basic math operations (`+`, `-`) to calculate real answers.

## 🚀 How to Run the Code

1. Download or clone this project folder to your computer.
2. Navigate to the `dist` folder.
3. Double-click the `index.html` file to open it in any web browser (Chrome, Safari, Edge, etc.) to see the live webpage.
4. Open both `index.html` and `script.js` in your code editor (like VS Code or Notepad).
5. Click the buttons on the webpage to see what they do, then look at the code to figure out *how* they do it!

## 🧑‍💻 Student Exploration Challenges

If you scroll to the very bottom of the `script.js` file, you will see your final missions! Once you understand how the `addNumbers()` and `subtractNumbers()` functions work, try these challenges:

1. **Multiply & Divide:** Write the missing JavaScript functions for `multiplyNumbers()` and `divideNumbers()`. *(Hint: In JavaScript, multiply is `*` and divide is `/`)*
2. **The 3-Number Input Challenge:** Can you update the HTML to include a third input box (`num3`), and update your JavaScript to calculate the math for all three numbers at once?
