# 🧮 Mission: The Advanced Calculator

*Created by Ms. Liu for Computer Science.*

Welcome to the Advanced Calculator project! In this mission, we are leveling up our basic calculator to handle more complex mathematical operations and learning how to protect our code from critical math errors.

## 🎯 What You Will Learn

This project introduces several vital Computer Science and Mathematics concepts:
* **Advanced Operators:** Using JavaScript to calculate Exponents (`**`) and Modulo/Remainders (`%`).
* **Integer Division:** Combining standard division (`/`) with `Math.floor()` to calculate how many whole times a number divides into another.
* **Edge Case Handling:** Using `if` statements to build "Error Traps" that stop the computer from trying to divide by zero.
* **State Management:** Using a custom `clearScreen()` function to reset the HTML inputs and text colors back to their default state.

## 🚀 How to Run the Code

1. Download or clone this project folder to your local computer.
2. Double-click the `index.html` file to open it in your web browser.
3. Open `script.js` in your code editor to see the logic.
4. Try typing `0` as your Second Number and hitting the Divide or Modulo buttons to test the Error Traps!

## 🧑‍💻 Student Exploration Challenges

If you look inside the `script.js` file, you will see a few comments left for you. Once you understand how the current code works, try this major challenge:

* **The 3-Number Input Challenge:** Right now, the calculator only handles `num1` and `num2`. Can you update the HTML to include a third input box (`num3`), and update your JavaScript functions to add, subtract, or multiply all three numbers at the same time?
