# 📐 Mission: The Master Unit Converter

*Created by Ms. Liu for Computer Science.*

Welcome to the Master Unit Converter project! In this mission, we are moving beyond simple calculators to build a highly practical tool that translates real-world measurements. [cite_start]You can view the original CodePen for this project here: [https://codepen.io/Hilda-Liu/pen/dPpoWmm](https://codepen.io/Hilda-Liu/pen/dPpoWmm)[cite: 1].

## 🎯 What You Will Learn

This project bridges algebraic formulas with JavaScript, teaching several vital concepts:
* **Inverse Mathematical Operations:** You will learn how to write a formula in one direction (e.g., Celsius to Fahrenheit), and then use algebra to reverse the Order of Operations for the opposite direction!
* **DOM Targeting:** Managing 6 different HTML `<input>` boxes and routing their data to 3 specific output paragraphs without mixing them up.
* **Floating-Point Precision:** Using the `.toFixed(2)` method to clean up messy, infinite decimals so your answers look professional.
* **Code Organization:** Using comments (`// --- SECTION ---`) to keep a growing JavaScript file neat and readable.

## 🚀 How to Run the Code

1. Download or clone this project folder to your local computer.
2. Navigate to the `dist` folder.
3. Double-click the `index.html` file to open the application in your web browser.
4. Open the `script.js` file in your code editor to see how the mathematical formulas are translated into code.
5. Type numbers into the input boxes and click the buttons to test the conversions!

## 🧑‍💻 Student Exploration Challenges

Ready to expand your converter? Try adding these new features:

1. **The Volume Converter:** Create a 4th section for Volume. Write the HTML and JavaScript to convert **Gallons to Liters** (Formula: `Liters = Gallons * 3.78541`), and **Liters back to Gallons**.
2. **The "Clear All" Button:** Write a single JavaScript function that clears the text out of *all 6* input boxes and resets all 3 result paragraphs back to `Result: ???`.
3. **UI Designer:** Open a `<style>` tag or attach a CSS file to make your converter look like a modern mobile app!
